
# 🌟 TechLumin

**Illuminating the Future of Tech** 🚀

TechLumin is a sleek, responsive platform designed to explore software engineering, data culture, and emerging technologies shaping our digital future. 💻📡📈

---

## ✨ Features

- 📚 **Articles**: In-depth writeups on tech topics
- 📊 **Dashboards**: Interactive data visualizations
- 🗂️ **Topics**: Categorized for easy navigation
- 🌗 **Dark/Light Mode**: Seamless theme switching
- 📱 **Responsive Design**: Mobile-first, device-ready
- 🔐 **User Accounts**: Favorites, profiles, personalization

---

## 📄 Article Pages

- 🤖 [AI/ML](ai-ml.html) – Artificial Intelligence & Machine Learning  
- ☁️ [Cloud Computing](cloud-comupting.html) – Cloud architecture & tools  
- 🔒 [Cybersecurity](cybersecurity.html) – Securing systems & data  
- 🛠️ [Data Engineering](data-engineering.html) – Pipelines, ETL, storage  
- 💻 [Software Development](software-development.html) – Best coding practices  
- 🏠 [Home](index.html) – Landing page for featured content  

---

## 💻 Technologies Used

- 🔧 HTML5, CSS3, JavaScript (ES6+)
- 🎨 [Tailwind CSS](https://tailwindcss.com/) – Utility-first styling
- 📈 [Chart.js](https://www.chartjs.org/) – Graphs and data viz
- ✨ [Font Awesome](https://fontawesome.com/) – Iconography
- 🌙 Built-in dark mode

---

## 🌐 Pages Overview

- 🏠 **Home** – Welcome page with highlights
- 📝 **Articles** – All content in one place
- 📊 **Dashboards** – Visual, interactive data stories
- 🏷️ **Topics** – Tech categories and tags
- 👤 **Profile** – User account & preferences
- 🔑 **Auth** – Login and register

## 🚦 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/techlumin.git
   cd techlumin
   ```

2. Open `index.html` in a browser
3. Browse and explore the platform
